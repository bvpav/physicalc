За нас

Какво е PhysiCalc?

PhysiCalc е сайт с калкулатори по физика. Надявахме се това да си проличи от името...

Този сайт го направихме като годишен проект по ИТ за 8 клас. Идеята беше да използваме и трите златни WEB технологии: HTML, CSS и JS.

Кои сте вие?

Ние сме Божидар Павлов, Калоян Атанасов и Никола Сачков от в клас. Ние сме ученици :)

и да.. това е всичко за нас basically

Използвани технологии

За създаването на PhysiCalc използвахме чисти HTML, CSS и JS без библиотеки, платформи и framewor-ци, освен Bulma CSS за готовите CSS стилове. Ако не беше Bulma, сайта щеше да изглежда много по-грозен.

Също така взехме икони от FontAwesome.

Изтеглихме снимки от G*ogle Images, дано не нарушаваме авторските им права...

Код и имплементация

Този сайт разчита на JavaScript за да работи. Всичките калкулатори правят изчисленията в браузера с JavaScript, без необходимост от сървър. За съжаление това означава, че този сайт няма да работи в по-стари браузери или в тези, които блокират JavaScript.

Кода изцяло е достъпен за преглед и публикуван като Свободен софтуер под GPL-3.0 лиценз. Можете да разгледате кода в GitHub .
